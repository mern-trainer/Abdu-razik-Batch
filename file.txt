Hello World!

-> github -> cloud storage. 
-> git -> version control system

-> git init -> initialize git in the project.
-> git add . -> add all files to git.
-> git add <file> <file> <file> -> add specific files to git.

-> git config --global user.name "<name>"
-> git config --global user.email "<email>"

-> git commit -m "first commit" -> commit the changes.
-> git remote add <remote_name> <remote_url> -> add remote repository.
-> git branch -M main -> change branch name to main.

-> git push <remote_name> <branch_name> -> push the changes to remote repository.
